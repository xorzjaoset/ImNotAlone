Template customization

1.How do I open the template?

The template is 100% ready to use, with no compilation needed. Just unzip, go to the html folder and open the index.html file in your browser using a development server

2.How do I install/uninstall a dependency with Bower?

Run bower install <dependency-name> --save. After this then you should add the installed dependencies to the .html files.

3.How do I edit the style.css file

you can simply open the style.css file into editor and change fonts colors etc.

3.How do I edit the sass files

First you need the gulpjs setup for compile the sass files into css
Here the simple and easy way to learn gulp
https://css-tricks.com/gulp-for-beginners/

4.How do I edit the email id to contact form

First make sure your website is runing using online server, contact form is working with only online server
1 - Open the finder/app/contact-form/process-contact.phph file into editor
2 - find this line into above file - $my_email = 'support@example.com';
3 - Place your email id and it will work
